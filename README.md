# 📊 Data Analysis with Python: freeCodeCamp Certification

This repository contains the projects I completed to earn the **Data Analysis with Python Certification** from freeCodeCamp. Each project focuses on a different aspect of data manipulation, statistical analysis, and visualization.

## 🎓 Certification Overview
Through this curriculum, I mastered the fundamental tools used in data science to extract insights from raw data. These projects laid the groundwork for my current transition into **Cheminformatics** and **Machine Learning**.

---

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Environment:** Jupyter Notebooks

---

## 📂 Project Gallery

### 1. Medical Data Visualizer
* **Goal:** Explore and visualize a dataset of medical examination results to find relationships between cardiac disease and lifestyle variables.
* **Key Skills:** Data cleaning, creating "long-form" data for categorical plots, and generating **Correlation Heatmaps**.

### 2. Time Series Visualizer
* **Goal:** Analyze a dataset of daily page views on the freeCodeCamp.org forum.
* **Key Skills:** Handling date/time objects in Pandas, box plots for seasonal trends, and line charts for growth patterns over multiple years.

### 3. Sea Level Predictor
* **Goal:** Analyze historical sea level change data and predict future changes through 2050.
* **Key Skills:** **Linear Regression** (using Scipy), scatter plots, and trend line extrapolation.

### 4. Demographic Data Analyzer
* **Goal:** Answer specific questions about a population dataset using Pandas indexing and descriptive statistics.
* **Key Skills:** Calculating percentages, finding maximum/minimum values within subgroups, and handling missing data.

### 5. Mean-Variance-Standard Deviation Calculator
* **Goal:** Create a function that uses NumPy to output the mean, variance, and standard deviation of the rows, columns, and elements in a 3x3 matrix.
